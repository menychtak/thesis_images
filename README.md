# Thesis Images

This repository contains image assets used in the author's thesis. It provides a stable, publicly accessible location for figures and related media to support reproducible builds and to avoid manual re-uploading across environments.

## Contents
- Figures, diagrams, and photographic material used in the thesis.

## Usage
Clone the repository into your thesis project or add it as a git submodule, then reference images via relative paths. Example commands:
- git clone https://github.com/menychtak/thesis_images.git
- git submodule add https://github.com/menychtak/thesis_images.git images
