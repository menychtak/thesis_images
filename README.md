# Thesis Images and Experiment Results

This repository was initially created to store image assets used in the author’s thesis. Its scope has since been extended to also store and organize experiment outputs, enabling systematic tracking of the experiments performed throughout the thesis.

Results are organized by date and, where applicable, by the notebook used to generate them. This structure helps preserve the relationship between notebooks, configurations, generated images, evaluation outputs, and experiment iterations, supporting reproducibility and easier comparison across runs.

## Contents

* Figures, diagrams, and photographic material used in the thesis.
* Generated images and other outputs produced during experiments.
* Experiment results organized by execution date and source notebook.
* Supporting metadata or files needed to trace and compare experimental runs.

## Usage

Clone the repository into your thesis project or add it as a Git submodule, then reference files using relative paths.

Example commands:

* `git clone https://github.com/menychtak/thesis_images.git`
* `git submodule add https://github.com/menychtak/thesis_images.git images`

When adding experiment outputs, organize them by date and notebook so that each result can be linked back to the experiment that produced it.

## Purpose

The repository provides a stable and publicly accessible location for thesis figures and experiment outputs. It supports reproducible builds, avoids repeated manual uploads across environments, and maintains a traceable history of the experiments conducted during the thesis.
